Jelajah Wisata

Website ini adalah portal pariwisata berkelanjutan Nusantara yang dirancang dengan HTML, CSS, dan Javascript.

Cara Menjalankan Project :

Metode 1: Buka Langsung di Browser
1. Buka folder project hasil ekstraksi.
2. Cari file `index.html` di folder.
3. Klik kanan `index.html` -> Pilih "Open With"-> Pilih browser Anda (Chrome, Edge, Firefox, dll.).
4. Website siap dijalankan dan dijelajahi secara interaktif!

Metode 2: Menggunakan Local Web Server
Untuk performa pemuatan aset yang paling optimal, Disarankan menggunakan local web server:
VS Code Live Server: Klik kanan `index.html` dan pilih "Open with Live Server".
